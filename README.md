# Anderlechtse NGO

## Beschrijving

Anderlechtse NGO is een webapplicatie voor een lokale organisatie in Anderlecht.

Met deze applicatie kunnen bezoekers:

* evenementen bekijken
* details van evenementen raadplegen
* nieuwe evenementen toevoegen
* informatie over de organisatie lezen
* contact opnemen via een contactformulier

## Pagina's

### Home

Overzicht van de beschikbare evenementen.

### Details

Toont extra informatie over een evenement.

### Nieuw evenement

Formulier om een nieuw evenement toe te voegen.

### About

Informatie over de NGO en haar missie.

### Contact

Contactformulier voor bezoekers.

## Technologieën

* Java
* Spring Boot
* HTML
* CSS

## Gebruikte hulpmiddelen

Tijdens de ontwikkeling van dit project werd gebruikgemaakt van AI-tools als ondersteuning voor:

* uitleg van programmeerconcepten;
* begeleiding bij het oplossen van fouten;
* verbetering van HTML- en CSS-code;
* controle van spelling en formulering.

Alle code werd begrepen, aangepast en geïntegreerd door de student.
